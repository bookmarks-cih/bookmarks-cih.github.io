Admin guide

      
How to use this panel and how blocks map to production files.

    

  

  

    
Workflow

    

      

        
1

        
Choose a block type
Select Job entry, File entry, or Creator card from the Block generator.

      

      

        
2

        
Fill in the fields
Each block type has specific fields. Required fields are marked. All data stays local in your browser.

      

      

        
3

        
Add to drafts
Click Add to drafts. You can queue multiple blocks before generating the final output.

      

      

        
4

        
Go to Generated code
Switch to the Output tab. Blocks are grouped by type. Copy each group and paste into the matching source file on GitHub.

Tips

      

        
Use Export to save your workspace as JSON before closing the browser.

All data is stored in localStorage. Nothing is sent to any server.
